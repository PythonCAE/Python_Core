# Adding One Element:
We can add a new element ton set using add() method.

syntax:

set_name.add(new_element) 

```python 
a = {1,2,3,"Hello",4}
a.add(5)
print(a)
```
### OutPut:
```
{1,2,3,"Helllo",4,5}
{1,2,3,"Helllo",5,4}
{"Helllo",4,5,1,3,2}
etc.....
```
